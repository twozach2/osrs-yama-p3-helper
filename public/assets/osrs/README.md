# Local OSRS Asset Drop Folder

Put locally converted OSRS `.glb` or `.gltf` files here.

Do not commit Jagex-owned models, textures, cache dumps, or the real `manifest.json`.
The app will load `public/assets/osrs/manifest.json` when it exists and otherwise use
the built-in primitive fallback models.

Use `manifest.example.json` as the template.
